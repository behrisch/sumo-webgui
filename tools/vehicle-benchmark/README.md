# Vehicle render benchmark

## Prerequisites

| Dependency | Version | Linux                | macOS               | Windows |
|------------|---------|----------------------|---------------------|---------|
| Node.js    |     18+ | `apt install nodejs` | `brew install node` | `winget install OpenJS.NodeJS` |
| npm        |      9+ | `apt install npm`    | bundled with Node   | bundled with Node |

## Running

```bash
npm install
npm run dev
```

Then open the browser (usually at http://localhost:5173/ but the terminal will tell you more) and compare the frame rates.
